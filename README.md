# news-shopping-crawling-homepage
Daum 뉴스 및 네이버쇼핑 크롤링 홈페이지
* 'Daum' 뉴스 검색 결과 데이터 수집
* 수집한 데이터를 웹페이지에 나타내고 엑셀 파일로도 저장
* 네이버 쇼핑 검색 결과 데이터 수집 (selenium을 이용한 동적 크롤링)
* 수집한 데이터를 웹페이지에 나타냄
